# NexusML – Automated Model Lifecycle Management Platform

NexusML – Automated Model Lifecycle Management Platform is an end-to-end MLOps project that automates data ingestion, validation, transformation, model training, evaluation, and deployment for employee attrition prediction using FastAPI and Scikit-Learn.
